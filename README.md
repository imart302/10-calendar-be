# Calendar App Backend

This is a demo app of a calendar services made with
  - Node
  - Express
  - CORS
  - Express-Validator
  - Mongoose


## Requeriments

Needs a mongodb server running or an repository made in Atlas

> [!NOTE]
> Setup your env variables as in env.example

## Dev

Run `npm run dev`

## Start

Run `npm run start`
